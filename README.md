# RetailOS — Professional POS & Retail Management System

A complete, production-ready Point of Sale and Retail Management 
System built in Python. Designed for small to medium retail shops.

## Key Features

- Barcode-driven Point of Sale (POS)
- Inventory Management with low-stock alerts
- Retail + Wholesale pricing support
- Purchase & Supplier Management
- Expense Tracking
- Advanced Reports (Balance Sheet, ABC Analysis, Tax, Cash Flow)
- Multi-user Login with history
- Receipt Printing + PDF Export
- Excel Export
- Built-in Trial + Lifetime Activation System
- Professional Dark UI (PySide6/PyQt6)
- Zero monthly costs — fully offline

## Tech Stack

- Python 3
- PySide6 / PyQt6
- SQLite (7 databases)
- openpyxl

## Target Market

Small retail shops, general stores, pharmacies, 
and kiryana stores — especially in South Asia.
